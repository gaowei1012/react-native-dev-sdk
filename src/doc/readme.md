> ReactNative Dev SDK

### 安装依赖

> 使用，安装需要将 dev 嵌入项目

```
  npm i @react-native-async-storage/async-storage @react-navigation/native axios react-native-root-toast --save
```

> 在项目 ios 目录下，执行 `pod isntall`
